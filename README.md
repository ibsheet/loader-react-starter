# ibsheet-loader-react-starter

[`@ibsheet/loader`](https://github.com/ibsheet/loader) starter for react

## clone project

```bash
git clone https://github.com/ibsheet/loader-react-starter ibl-react-starter
cd ibl-react-starter
```

## setup and start

#### Using npm:

```
npm install
npm start
```

#### Using yarn:

```
yarn
yarn start
```
